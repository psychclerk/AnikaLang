Math Utilities

mathutils.fms

Advanced math helpers for everyday calculations — percentages, clamping, interpolation, unit conversions, financial math, and number formatting. All functions handle edge cases gracefully and return sensible defaults.

💡
How to Import
Add this line at the top of your script:
INCLUDE "../../includes/mathutils.fms"

% Percentages & Ratios 6 functions

FunctionArgumentsReturnsDescription
PERCENT_OF (value, percent) float Calculate percent% of value. Example: PERCENT_OF(200, 15) → 30
PERCENT_CHANGE (old, new) float Percentage change from old to new. Returns negative for decrease.
RATIO (a, b) float Ratio of a to b. Returns 0 if b is zero.
PROPORTION (part, whole) float Returns part/whole as a decimal (0–1). Safe division by zero.
MARKUP (cost, percent) float Apply markup percentage to cost. Example: MARKUP(100, 25) → 125
DISCOUNT (price, percent) float Apply discount percentage. Example: DISCOUNT(80, 20) → 64

📏 Number Range & Bounds 5 functions

FunctionArgumentsReturnsDescription
CLAMP (value, min, max) number Clamp value between min and max. Essential for game/UI values.
LERP (a, b, t) float Linear interpolation between a and b at t (0–1). Great for animations.
IN_RANGE (value, min, max) bool Returns TRUE if value is between min and max (inclusive).
WRAP (value, min, max) number Wrap value around range. Example: WRAP(370, 0, 360) → 10 (degrees).
SIGN (value) int Returns 1, -1, or 0 based on sign of value.

🎯 Rounding & Precision 5 functions

FunctionArgumentsReturnsDescription
ROUND_TO (value, step) float Round to nearest multiple of step. Example: ROUND_TO(7.3, 0.5) → 7.5
FIXED (value, decimals) string Format number as string with fixed decimal places. Example: FIXED(3.14159, 2) → "3.14"
TRUNCATE (value, decimals) float Truncate (not round) to N decimals. Example: TRUNCATE(3.999, 2) → 3.99
NEAREST_INT (value) int Round to nearest integer (banker's rounding).
APPROX_EQUAL (a, b, [tolerance]) bool Check if two floats are approximately equal. Default tolerance: 0.0001.

💰 Financial Math 5 functions

FunctionArgumentsReturnsDescription
SIMPLE_INTEREST (principal, rate, years) float Calculate simple interest. rate is percentage per year.
COMPOUND_INTEREST (principal, rate, years, [n]) float Compound interest. n = compounding periods per year (default 12).
TIP (bill, percent) float Calculate tip amount. Example: TIP(50, 18) → 9
TAX (amount, rate) float Calculate tax amount. Example: TAX(100, 8.5) → 8.5
EMI (principal, rate, months) float Monthly EMI (Equated Monthly Installment) for loans.

📐 Geometry 6 functions

FunctionArgumentsReturnsDescription
DISTANCE (x1, y1, x2, y2) float Euclidean distance between two 2D points.
DISTANCE_3D (x1, y1, z1, x2, y2, z2) float Euclidean distance between two 3D points.
ANGLE_BETWEEN (x1, y1, x2, y2) float Angle (in degrees) between two points relative to origin.
CIRCLE_AREA (radius) float Area of circle: π × r²
RECTANGLE_AREA (width, height) float Area of rectangle.
TRIANGLE_AREA (base, height) float Area of triangle: ½ × base × height.

🔄 Unit Conversions 6 functions

FunctionArgumentsReturnsDescription
CELSIUS_TO_F (c) float Convert Celsius to Fahrenheit.
F_TO_CELSIUS (f) float Convert Fahrenheit to Celsius.
KM_TO_MILES (km) float Convert kilometers to miles.
MILES_TO_KM (miles) float Convert miles to kilometers.
KG_TO_LBS (kg) float Convert kilograms to pounds.
LBS_TO_KG (lbs) float Convert pounds to kilograms.

✨ Number Formatting 5 functions

FunctionArgumentsReturnsDescription
FORMAT_NUMBER (value, [decimals]) string Format with thousand separators. FORMAT_NUMBER(1234567, 2) → "1,234,567.00"
FORMAT_CURRENCY (value, [symbol]) string Format as currency. Default symbol: "$". Example: FORMAT_CURRENCY(1234.5) → "$1,234.50"
FORMAT_BYTES (bytes) string Human-readable bytes. FORMAT_BYTES(1536) → "1.5 KB"
FORMAT_COMPACT (value) string Compact notation. FORMAT_COMPACT(1500000) → "1.5M"
ROMAN_NUMERAL (number) string Convert integer (1–3999) to Roman numeral. ROMAN_NUMERAL(2024) → "MMXXIV"
mathutils_example.fms
1INCLUDE "../../includes/mathutils.fms"
2
3# Percentage calculations
4SET price TO 100
5SET discounted TO DISCOUNT(price, 20) # 80
6SET with_tax TO TAX(discounted, 8.5) # 86.8
7
8# Clamping a game value
9SET health TO CLAMP(150, 0, 100) # 100
10
11# Format for display
12SET display TO FORMAT_CURRENCY(with_tax) # "$86.80"
13UI_ALERT("Final price: " & display)

String Utilities

stringutils.fms

Powerful string manipulation helpers — case conversion, padding, slug generation, template formatting, validation, encoding, and advanced search/replace. Perfect for data processing, URL building, and user input sanitization.

💡
How to Import
Add this line at the top of your script:
INCLUDE "../../includes/stringutils.fms"

🔤 Case Conversion 6 functions

FunctionArgumentsReturnsDescription
TITLE_CASE (text) string Capitalize first letter of each word. TITLE_CASE("hello world") → "Hello World"
SENTENCE_CASE (text) string Capitalize first letter of each sentence.
CAMEL_CASE (text) string Convert to camelCase. CAMEL_CASE("hello world") → "helloWorld"
SNAKE_CASE (text) string Convert to snake_case. SNAKE_CASE("Hello World") → "hello_world"
KEBAB_CASE (text) string Convert to kebab-case. KEBAB_CASE("Hello World") → "hello-world"
INVERT_CASE (text) string Invert case of each character. INVERT_CASE("Hello") → "hELLO"

📐 Padding & Truncation 5 functions

FunctionArgumentsReturnsDescription
PAD_LEFT (text, length, [char]) string Pad on left. Default char: space. PAD_LEFT("42", 5, "0") → "00042"
PAD_RIGHT (text, length, [char]) string Pad on right. Default char: space.
CENTER (text, length, [char]) string Center text within length. Default char: space.
TRUNCATE (text, length, [suffix]) string Truncate to length with suffix. Default suffix: "..."
REPEAT (text, count) string Repeat text N times. REPEAT("ab", 3) → "ababab"

🔍 Search & Replace 5 functions

FunctionArgumentsReturnsDescription
COUNT_OCCURRENCES (text, substr) int Count non-overlapping occurrences of substr in text.
REPLACE_FIRST (text, old, new) string Replace only the first occurrence.
REPLACE_LAST (text, old, new) string Replace only the last occurrence.
REPLACE_IGNORE_CASE (text, old, new) string Case-insensitive replace all.
MASK (text, [char]) string Mask all characters except last 4. Default char: "*". Useful for credit cards.

✂️ Split & Join 5 functions

FunctionArgumentsReturnsDescription
WORDS (text) list Split text into list of words (splits on any whitespace).
LINES (text) list Split text into list of lines.
CHARS (text) list Split text into list of individual characters.
JOIN_LIST (list, delimiter) string Join list into string with delimiter. JOIN_LIST(["a","b","c"], ",") → "a,b,c"
CHUNK (text, size) list Split text into chunks of N characters.

✅ Validation 6 functions

FunctionArgumentsReturnsDescription
IS_EMAIL (text) bool Check if text is a valid email address.
IS_URL (text) bool Check if text is a valid URL (http/https/ftp).
IS_NUMERIC (text) bool Check if text can be parsed as a number.
IS_ALPHA (text) bool Check if text contains only letters.
IS_ALPHANUM (text) bool Check if text contains only letters and digits.
IS_PALINDROME (text) bool Check if text reads the same forwards and backwards (case-insensitive).

📊 Text Analysis 5 functions

FunctionArgumentsReturnsDescription
WORD_COUNT (text) int Count words in text.
LINE_COUNT (text) int Count lines in text.
SENTENCE_COUNT (text) int Count sentences (splits on ., !, ?).
CHAR_FREQUENCY (text) dict Returns dict of character → count.
MOST_COMMON_WORD (text) string Returns the most frequently occurring word.

🔗 URL & Slug Generation 4 functions

FunctionArgumentsReturnsDescription
SLUGIFY (text) string Convert to URL-safe slug. SLUGIFY("Hello World!") → "hello-world"
URL_ENCODE (text) string URL-encode special characters.
URL_DECODE (text) string Decode URL-encoded string.
EXTRACT_DOMAIN (url) string Extract domain from URL. EXTRACT_DOMAIN("https://example.com/path") → "example.com"

📋 Template & Formatting 3 functions

FunctionArgumentsReturnsDescription
TEMPLATE (template, dict) string Replace {{key}} placeholders with values from dict.
PLURALIZE (word, count, [plural]) string Return singular or plural form. Auto-adds "s" if plural not given.
ORDINAL (number) string Convert number to ordinal string. ORDINAL(3) → "3rd"
stringutils_example.fms
1INCLUDE "../../includes/stringutils.fms"
2
3# Case conversion
4SET name TO TITLE_CASE("john doe") # "John Doe"
5SET var_name TO SNAKE_CASE("user name") # "user_name"
6
7# URL generation
8SET title TO "Hello World! This is Cool"
9SET slug TO SLUGIFY(title) # "hello-world-this-is-cool"
10
11# Validation
12IF IS_EMAIL("user@example.com") THEN
13 UI_ALERT("Valid email!")
14END IF
15
16# Template substitution
17SET data TO {"name": "Alice", "age": "30"}
18SET msg TO TEMPLATE("Hi {{name}}, you are {{age}}!", data)
19UI_ALERT(msg) # "Hi Alice, you are 30!"

Pro Tip
Both modules are pure AnikaLang — no external dependencies required! They're built using only the standard library functions, so they work everywhere AnikaLang runs.

Note on Namespaces
All functions are added to the global namespace when you INCLUDE the module. If you include both modules, you get access to all 42 functions directly — no prefix needed.
