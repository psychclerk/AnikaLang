See It In Action

Readable. Powerful. Real.

A complete desktop application in just a few lines. No boilerplate, no complexity.

counter_app.fms
1# A complete counter application
2SET count TO 0
3SET display TO NULL
4
5FUNCTION increment()
6 SET count TO count + 1
7 UI_SET(display, "Count: " & STR(count))
8END FUNCTION
9
10# Build the UI
11SET win TO UI_WINDOW("Counter App", 300, 200)
12SET display TO UI_LABEL(win, "Count: 0", 100, 50, 100, 30)
13UI_BUTTON(win, "+1", "increment", 110, 100, 80, 30)
14
15UI_MAINLOOP(win)

Get Started

Up and Running in 30 Seconds

No Python needed. No dependencies to install. Just download and run.

1

Download from GitHub

github.com/psychclerk/anikalang/anikawx.zip

Download AnikaLang.exe for Windows

2

Extract to a folder like c:\anikawx\

anikawx.exe

Double-click anikawx.py launch the interpreter. Run the IDE from the projects folder.

3

Create your first script

SET win TO UI_WINDOW("Hello", 300, 200)

Save as hello.fms

4

Run your script

anikawx.exe hello.fms

Or run it from the IDE
